1 Download and Import Data

  1. Download the datasets china.csv (right click here), nba_players_19.csv (right click here), mlb.csv (right click here), and drug_use.csv (right click here) and save them in the data folder.

  2. Import the package tidyverse.

  3. Read the four datasets and save them as china, nba, mlb, and drug.

2 Childcare Hours in China

The dataset china.csv records childcare time and the gender and education of each respondent. By browsing the data, you might immediately notice that some values in the dataset appear unusual. For example, some values in the variable child_care are -99, which indicates that the observation is missing.

Use the filter() function to keep only the observations where child_care is not equal to -99. Assign the result to a variable called china_filter.

china_filter <- china %>%
  filter(child_care != -99)

3 NBA Players’ Height

The dataset nba_players_19.csv records information about NBA players in the 2018-2019 season. We would like to know who the tallest players in the NBA are. Follow the instructions step-by-step:

  1. Create a new variable called height_cm using the mutate() function, where height_cm = 2.54 * height, given that the unit of height is inches.

  2. Sort the dataset by height_cm in descending order.

  3. Use the select() function to keep only first_name, last_name, position, and height_cm.

Assign the result to a variable called nba_height. Using the pipeline %>% to chain all the operations together is strongly recommended. Try it!

nba_height <- nba %>%
  mutate(height_cm = 2.54 * height) %>%
  arrange(desc(height_cm)) %>%
  select(first_name, last_name, position, height_cm)

4 Summarize Your Data

The dataset mlb.csv contains salary data for Major League Baseball players in 2010.

We want to determine which team invests the most money in building a baseball team. Follow the steps below:

  1. Use the group_by() function to group the observations by team.

  2. Use the summarize() function to calculate the total salary spent by each team using sum(). Name the calculated column total_salary.

  3. Use the arrange() function to sort the observations by total_salary in descending order.

Assign the result to a variable called team_salary and print it.

team_salary <- mlb %>%
  group_by(team) %>%
  summarize(total_salary = sum(salary)) %>%
  arrange(desc(total_salary))

print(team_salary)

5 Contingency Table

The dataset drug_use.csv contains drug-use records for 445 student-parent pairs. Each student and parent reports whether they use drugs.

  1. Use the table() function to create a contingency table where the rows represent student and the columns represent parents. Assign the output to a variable called count_table.

  2. Use addmargins() to add the sum row and sum column. Assign the result to a variable called count_table_sum.

  3. Print count_table_sum to check the distribution.

count_table <- table(
  drug$student,
  drug$parents,
  dnn = c("Student's Drug Use", "Parent's Drug Use")
)

count_table_sum <- addmargins(count_table)
print(count_table_sum)
